Immich Photos for Gmail

Connect your self-hosted Immich library so you can attach photos directly from Gmail compose.
The HTTPS URL of your Immich instance, no trailing slash.
Stored locally in this browser only. Never sent anywhere except your Immich server.

Where do I get an API key?

  1. Open your Immich web app and sign in.
  2. Click your profile avatar, then Account Settings.
  3. Open the API Keys section, then New API Key.
  4. Copy the generated key and paste it above.
